oyster-seed
===========

Project initial template that based on oyster recommended structure to build node apps


## Prerequisites

- NodeJS `v4.2.4`
- npm `v3.7.3`
- nodemon `v1.9.1`
- MongoDB `v3.2.3`
- mysql  `Ver 14.14 Distrib 5.5.35`

## Setup Instructions

- Clone repository
- `$ ./setupServer`
- `$ npm start`
